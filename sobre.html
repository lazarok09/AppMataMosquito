Bem vindo ao Mata Mosquito

Nesse presente aplicativo foram utilizadas tecnologias simples em sua essência, porém, ao adentrar na codificação, cada vez mais pediam-se funções complexas. Ainda assim, o uso das funções nativas do javascript tornou o complexo em simples.

Aprendizados

  • Utilizar, criar e modificar elementos HTML através do JS
  • Variáveis dinâmicas por todo o código
  • Reutilização de função seguindo influência do clean code
  • Valores randomicos utilizando função nativa Math
  • Praticar responsividade criando pagina sobre & media queries
